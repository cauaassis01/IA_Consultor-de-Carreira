# IA_Consultor-de-Carreira
Este prompt transforma o ChatGPT em um consultor de carreiras que analisa as falas e os dados do aluno para identificar seus interesses e objetivos. Em seguida, gera uma orientação personalizada com resumo do perfil, áreas recomendadas, média salarial, caminhos de estudo e uma motivação final alinhada às características do estudante.
